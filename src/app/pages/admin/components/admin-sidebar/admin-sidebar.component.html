Administración

Como creador, es un placer que estés aquí para ayudar con la administración de bocetos, con la ayuda de todos estoy seguro de que crearemos un gran comunidad.

Tienes el poder para decidir que bocetos entran y cuáles no. Si eres administrador es por que eres alguien con criterio, tienes debilidad por el buen gusto y diseño, además de saber bastante de al menos un tema en concreto.

Cómo funciona

Utiliza la lista para elegir que boceto quieres administrar. Para que un boceto sea aprobado deberá cumplir una serie de requisitos que tú como administrador seguro ya sabes.

Utiliza tu criterio para determinar que requisitos cumple. Obviamente para saber esto hay que informarse sobre el boceto primero, leerlo por encima y decidir en base a ello.
